## Quick Reference

**Business Model**: Educational data analytics platform with subscription tiers and crypto-based feature unlocking
**Tech Stack**: Next.js (SSR) + Rust (Clean Architecture) + PostgreSQL + Firebase Auth
**Key Features**: Dynamic templates, admin assignments, crypto payments, real-time analytics
