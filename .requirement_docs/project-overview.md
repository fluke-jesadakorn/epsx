# EPSX Project Overview

## Executive Summary

EPSX is an educational market data analytics platform that democratizes algorithmic insights through AI-driven analysis. Users access sophisticated EPS growth-based algorithms and pattern recognition without requiring technical analysis expertise.

## Business Model

### Core Value Proposition
- **Algorithm-first approach** vs. traditional tool-first platforms
- **Educational positioning** with comprehensive compliance framework
- **No technical analysis knowledge required** for advanced insights
- **Dynamic feature unlocking** through subscription tiers and crypto payments

### Target Market
- **Primary**: Data analysts seeking algorithmic insights
- **Secondary**: Market data enthusiasts and analytics professionals
- **Goal**: 20,000+ active subscribers with scalable API integration

### Revenue Streams
1. **Subscription Tiers**: Bronze, Silver, Gold, Platinum with progressive feature access
2. **Crypto Pay-Per-Feature**: Individual feature purchases (USDT, USDC, ETH, BTC)
3. **Enterprise Solutions**: Custom permission profiles and white-label analytics
4. **API Access**: Programmatic access to proprietary algorithms

## Technical Foundation ✅ IMPLEMENTED

### Architecture Strategy ✅ OPERATIONAL
- **Frontend**: Next.js 15 with Server-Side Rendering (SSR-first) ✅
- **Backend**: Rust with Clean Architecture + Hexagonal patterns ✅
- **Database**: PostgreSQL schema designed, Firebase Auth operational ✅
- **Deployment**: Vercel with CI/CD pipeline and auto-scaling ✅

### Key Innovations ✅ FULLY OPERATIONAL & ENHANCED
- **Dynamic Permission Profile System**: Auto-assignment engine with granular access control ✅
  - API endpoint access control with wildcard pattern matching
  - Frontend route protection with middleware integration
  - Rate limiting based on permission profiles (Bronze: 100/hr, Silver: 500/hr, Gold: 2000/hr)
- **Payment-Based Auto-Assignment**: Automatic feature unlocking on payment completion ✅
  - Webhook-triggered permission activation
  - Configurable expiration dates (Monthly/Annual/Lifetime)
  - Automatic renewal notifications (7 days before expiry)
- **Admin Direct Assignment**: Enhanced dashboard with expiration management ✅
  - Time-limited trial assignments
  - Bulk operations with expiration dates
  - Complete audit trail with payment correlation
- **Educational Compliance**: Built-in disclaimers and audit trails ✅
- **Crypto Payment Integration**: MusePay with auto-assignment triggers ✅
