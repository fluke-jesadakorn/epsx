# About EPSX

EPSX is a web3 analytics and payments platform built on Binance Smart Chain.

## Mission

Make web3 commerce **as easy as web2** — predictable costs, programmable rules, and complete user control.

## Team

Built by a small team of engineers passionate about DeFi and stablecoin payments.

## Architecture

We use a microservices architecture:

- **Rust** for backend (Axum + SQLx + Alloy)
- **Solidity** for smart contracts (Foundry)
- **Dioxus 0.7** for SSR frontend
- **PostgreSQL** for storage
- **Redis** for events and caching
